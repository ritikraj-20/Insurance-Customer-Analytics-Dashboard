📊 Insurance Customer Analytics Dashboard
📌 Overview

This project analyzes insurance customer data using SQL and Power BI. It demonstrates an end-to-end data workflow, starting from data creation and storage in SQL to building an interactive dashboard in Power BI.

The objective is to generate insights on customer behavior, premium contributions, and claims activity to support data-driven decision-making.

⚙️ Tech Stack
SQL
Power BI
🔄 Workflow
1. Data Creation (SQL)
Created and structured tables for customers, policies, and claims
Stored and managed data using SQL
2. Data Integration
Connected SQL database to Power BI
Imported datasets for analysis
3. Data Cleaning & Transformation
Handled missing values
Standardized formats
Created calculated columns and measures
4. Data Modeling
Built relationships between tables
Designed a star schema
Optimized data model for performance
5. Dashboard Development
Developed interactive and user-friendly visualizations
Enabled filtering and drill-down analysis
📈 Key Features
Customer segmentation
Total premium analysis
Claims analysis (settled and pending)
Active vs deactivated customers
Interactive dashboard with filters and slicers
