Insurance Customer Analytics Dashboard
Overview

This project analyzes insurance customer data using SQL and Power BI. It demonstrates an end-to-end workflow from data creation and storage to building an interactive dashboard.

The goal is to generate insights on customer behavior, premium contributions, and claims activity.

Tech Stack
SQL
Power BI
Workflow

1. Data Creation (SQL)

Created structured tables for customers, policies, and claims
Stored and managed data using SQL

2. Data Integration

Connected SQL database to Power BI
Imported datasets for analysis

3. Data Cleaning & Transformation

Handled missing values
Standardized formats
Created calculated columns and measures

4. Data Modeling

Built relationships between tables
Designed a star schema

5. Dashboard Development

Created interactive dashboards
Added filters and drill-down features
Key Features
Customer segmentation
Total premium analysis
Claims analysis (settled and pending)
Active vs deactivated customers
Interactive visuals
